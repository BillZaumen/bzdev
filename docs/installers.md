# Installers
The bzdev installer must be run before any of the other installers are run,
with the exception of cvrdecode.

  - [bikeshr (libbikeshr-*)](https://billzaumen.github.io/bzdev/installers/bikeshr-install-1.4.8.jar)

  - [bzdev (libbzdev-*)](https://billzaumen.github.io/bzdev/installers/bzdev-install-1.4.5.jar)

  - [cvrdecode](https://billzaumen.github.io/bzdev/installers/cvrdecode-install-1.1.jar)

  - [epts](https://billzaumen.github.io/bzdev/installers/epts-install-0.1.33.jar)

  - [geth](https://billzaumen.github.io/bzdev/installers/geth-install-1.0.2.jar)

  - [roadanim (librdanim-*)](https://billzaumen.github.io/bzdev/installers/roadanim-install-1.4.9.jar)

  - [webnail](https://billzaumen.github.io/bzdev/installers/webnail-install-1.6.7.jar)

To run an installer, use the command

```

java -jar INSTALLER
```

where INSTALLER is a downloaded JAR file.
